1.- Mueve el archivo .rar o .zip al lugar/carpeta donde quieres instalar CReminders

2.- Extrae el archivo, te quedaras con un folder llamado CReminders

3.- Dentro del folder, corre CReminders.exe

4.- Modifica el archivo info.txt para agregar las notificaciones

----------------------------------------------------------------------------------

Recuerda que el archivo commands.exe te permite realizar varias funciones:

mostrar	
Muestra rápidamente como se verían los recordatorios

desactivar	
Elimina el programa del Registro y detiene su ejecución. Para volver a activarlo se debe ejecutar CReminders.exe de nuevo

reset es	
Resetea el archivo info.txt a su estado inicial, en español. Note que esto borrara todos los recordatorios que haya creado
